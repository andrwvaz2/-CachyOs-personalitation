# Cachy-Dots 🚀

Mi personalización personal para **CachyOS** con Hyprland. Este script automatiza la instalación de paquetes y la aplicación de mis dotfiles.

## 📦 Componentes
- **WM:** Hyprland
- **Bar:** Waybar
- **Terminal:** Ghostty / Kitty
- **Shell:** Fish
- **Lanzador:** Wofi
- **Wallpaper:** swww

## 🚀 Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/TU_USUARIO/Cachy-Dots.git
   cd Cachy-Dots
   ```

2. Dale permisos de ejecución al script:
   ```bash
   chmod +x setup.sh
   ```

3. Ejecuta el instalador:
   ```bash
   ./setup.sh
   ```

## ⚠️ Nota sobre Wallpapers
El script asume que tienes una carpeta llamada `~/Imágenes/`. Asegúrate de actualizar la ruta del wallpaper en `~/.config/hypr/hyprland.conf` o incluir tus fondos en la carpeta `assets/` de este repo.
